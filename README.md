| Type de branche | Quand l'utiliser ?                                                             | Exemple                 |
| --------------- | ------------------------------------------------------------------------------ | ----------------------- |
| **`feature/`**  | Pour développer une nouvelle fonctionnalité                                    | `feature/profil`        |
| **`bugfix/`**   | Pour corriger un bug détecté en environnement de développement                 | `bugfix/button-error`   |
| **`hotfix/`**   | Pour corriger rapidement un bug critique en production                         | `hotfix/security-issue` |
| **`release/`**  | Pour préparer une nouvelle version de l'application                            | `release/v1.0.0`        |
| **`refactor/`** | Pour améliorer, optimiser ou restructurer du code sans changer le comportement | `refactor/api`          |
| **`chore/`**    | Pour les tâches de maintenance ou mises à jour mineures                        | `chore/update-packages` |
| **`style/`**    | Pour corriger la mise en forme (indentation, CSS, conventions)                 | `style/fix-css`         |
| **`test/`**     | Pour créer ou modifier des tests ou expérimenter                               | `test/new-ui`           |
